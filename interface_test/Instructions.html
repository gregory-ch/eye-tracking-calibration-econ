Instructions

In this game, the asset value can be either 0 or 100.

Each participant sees:

  • Previous decisions
  • One of two possible signals (red or blue ball)

Signal distribution:

  • If asset value is 100:
    • 70% chance of red ball
    • 30% chance of blue ball
  • If asset value is 0:
    • 30% chance of red ball
    • 70% chance of blue ball

Cell information:

  • Decision number: top-left corner
  • Decision value: bottom-right corner
  • Your input: bottom-right in the input cell
  • Signal ball: bottom-right in the last cell
